# F12
An open source 12 key, 1 encoder, that can be entirely assembled using only through hole components, powered by Vial Firmware
![PCB](https://github.com/hadi-syafiq/F12/blob/main/6x2.png?raw=true)

| Diode 1n4148                | 13     | |
| 4,7u capacitor              | 1      | |
